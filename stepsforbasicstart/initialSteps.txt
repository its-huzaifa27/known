🚀 Full Stack Setup (from empty folder)
🔹 Step 1: Open your project folder

Open terminal inside your empty cloned folder

🔹 Step 2: Create main folders
mkdir client server

👉 Now you have:

client → frontend
server → backend
🔹 Step 3: Setup frontend (React + Vite)
npm create vite@latest client

👉 When asked:

Select: React
Select: JavaScript

Then:

cd client
npm install
cd ..
🔹 Step 4: Setup backend (Node + Express)
cd server
npm init -y

Install dependencies:

npm install express cors dotenv
npm install nodemon --save-dev

Go back:

cd ..
🔹 Step 5: Create backend entry file

Inside server folder, create file:

server/server.js

Add basic code:

const express = require("express");
const app = express();

app.use(express.json());

app.get("/", (req, res) => {
  res.send("API running...");
});

const PORT = 5000;
app.listen(PORT, () => console.log(`Server running on ${PORT}`));
🔹 Step 6: Add backend scripts

Open server/package.json and update:

"scripts": {
  "start": "node server.js",
  "dev": "nodemon server.js"
}
🔹 Step 7: Create .gitignore (IMPORTANT)

In root folder (main project), create:

.gitignore

Add:

node_modules
.env
dist
build
🔹 Step 8: Run backend
cd server
npm run dev

👉 Check: http://localhost:5000

🔹 Step 9: Run frontend

Open new terminal:

cd client
npm run dev

👉 Check: http://localhost:5173

🔹 Step 10: First Git commit (important habit)

Open GitHub Desktop

Write message: initial setup
Commit
Push
✅ Final Structure
your-project/
├── client/
├── server/
├── .gitignore
🔥 That’s it — base ready

Now you can start building your app:

Create task API (backend)
Connect frontend form
Add deadline logic
